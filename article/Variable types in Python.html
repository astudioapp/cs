about::Variable types in Python::15 Feb 2021::Sebastian Doe::/article/How to define a variable in Python::cs text::These are some common types of variables in Python: bold::Strings text::Strings store text, and they are defined by using double apostrophes around the value. text::For example, this is how you would define the text "a" to the variable letter: code::letter = "a" bold::Integers text::Integers are numbers which don't have a decimal point (i.e. are whole numbers). They are defined by just stating the number. text::For example, this is how you would define the number 42 to the variable number: code::number = 42 bold::Floats text::Floats are numbers with decimal points. They are again defined by just stating the number. text::For example, this is how you would define the number 1.67 to the variable number: code::number = 1.67 bold::Booleans text::Booleans are used to store two values, either True or False. They are defined by either stating True or False (without any apostrophes). text::For example, this is how you would define the value True to the variable positiveNumber: code::positiveNumber = True
