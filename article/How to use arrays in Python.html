about::How to use arrays in Python::15 Feb 2021::Sebastian Doe::/article/How to define a variable in Python::cs bold::Defining an array text::To define an array, you need to put the variable's name, an equal sign, then the array. An example of an array is below. code::array = ["a", "b", "c"] text::The array is surrounded by square brackets, and inside there are values surrounded by double brackets and separated by commas. bold::Reading from an array text::To read from an array, enter the name of the array and the position in square brackets (remember that in Python, counting starts from 0). text::For example, this is how to get letter "a" from the array: code::print(array[0]) # Prints "a" bold::Replacing a value in the array text::To replace a value in the array, enter the name of the array and the position in square brackets, an equal sign, then the value you want it to be replaced with. text::For example, this is how to replace the letter "c" with the letter "x" in the above array: code::array[2] = "x" text::You can then print the array to check that the value has been replaced. code::print(array) # Prints ["a", "b", "x"] bold::Adding a value to the array text::To add a value to the end of an array, use the append() function, as follows: code::array.append("y") text::The value of the array is now ["a", "b", "x", "y"]. bold::Removing a value from an array text::To remove a value from an array, use the function pop(index), replacing index with the position in the array. text::For example, here's how you can remove the value "x" from the array: code::array.pop(2) text::The value of the array is now ["a", "b", "y"].
